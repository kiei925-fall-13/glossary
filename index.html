KIEI925 Glossary

I don't think that word means what you think it means.

class
category
type
A general blueprint for a kind of thing. Examples include: String, Array, Fixnum.
object
instance
thing
noun
An individual member of a class. Examples include: the string"howdy", the array [1, 9, 4], and the fixnum 704.
method
instruction
verb
An action to be performed by an object. Examples include: String#capitalize, Array#length, and Fixnum#odd?.
argument
input
An input to a method used to customize its behavior. Arguments to methods are often required, but are sometimes optional. Examples: 
puts                   # No argument provided, displays a blank line
puts("Howdy")          # Displays our message
"Hello".gsub("l", "w") # => "Hewwo"
3.+(2)                 # => 5
list
Used informally to refer to arrays and hashes.
array
collection
A data structure most often used to store an arbitrary number of similar elements. Elements are retrieved by position. Example: 
a = ["Josh", "Jeff", "Raghu"]
a[1] # => "Jeff"
hash
key/value pairs
options
dictionary
A data structure most often used to store a known list of distinct properties. Elements are retrieved by key. Example: 
h = [:first => "Raghu", :last => "Betina"]
a[:last] # => "Betina"
